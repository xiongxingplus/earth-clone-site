# 参考资料

本页收集克隆地球项目开发过程中参考的技术资料、论文与开源项目。

## 数字地球相关

- [Google Earth](https://earth.google.com/) — 商业数字地球产品
- [Cesium](https://cesium.com/) — 开源 3D 地理可视化平台
- [OpenStreetMap](https://www.openstreetmap.org/) — 开放地图数据

## 引擎与渲染

- [Unreal Engine 5](https://www.unrealengine.com/) — 项目使用的渲染引擎
- Nanite / Virtual Shadow Maps — UE5 核心渲染技术

## 地理数据

- [SRTM](https://www.usgs.gov/centers/eros/science/usgs-eros-archive-digital-elevation-shuttle-radar-topography-mission-srtm-1) — NASA 全球高程数据
- [Overpass API](https://overpass-turbo.eu/) — OSM 数据查询接口

## 四叉树与流式加载

- QuadTree spatial indexing
- Mercator projection tiling schemes
- LOD（Level of Detail）算法
