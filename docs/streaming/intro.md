# 流式加载

流式加载系统是克隆地球的核心基础设施，负责按需加载与卸载地理数据瓦片。

## 概述

全球地理数据量巨大，无法一次性加载。流式加载系统基于自研的 QuadTree（四叉树）架构，根据相机位置和视野范围，动态加载所需的地理瓦片。

## 关键技术

- **QuadTree 结构**：全球数据按四叉树组织，支持多级缩放
- **视锥裁剪**：只加载视野内的瓦片
- **优先级队列**：近处瓦片优先加载
- **异步加载**：不阻塞主渲染线程

## 设计目标

- 平滑的缩放体验，无明显加载卡顿
- 内存可控，及时卸载远处瓦片
- 支持多种数据类型（地形、建筑、道路）的统一加载
