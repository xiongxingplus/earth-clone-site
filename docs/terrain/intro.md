# 地形系统

地形系统是克隆地球的基础模块，负责全球地形的渲染与管理。

## 概述

地形系统基于真实高程数据（DEM），将全球地表划分为四叉树瓦片，实现多层级细节（LOD）渲染。

## 关键技术

- **数据源**：SRTM / ASTER 等公开高程数据集
- **瓦片划分**：Mercator 投影下的四叉树分块
- **LOD 策略**：基于相机距离的动态细节调整
- **法线计算**：从高程数据实时生成地形法线

## 数据流程

```
DEM 数据 → 瓦片切割 → LOD 层级生成 → 引擎加载 → 实时渲染
```
