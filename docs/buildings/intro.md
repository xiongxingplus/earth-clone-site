# 建筑系统

建筑系统负责将真实世界的建筑数据转化为三维模型，放置在数字地球的地形之上。

## 概述

通过解析 OpenStreetMap 的建筑轮廓数据，生成带有高度信息的三维建筑模型。

## 关键技术

- **数据源**：OpenStreetMap 建筑轮廓（building footprints）
- **高度推断**：基于 OSM 标签或统计模型估算建筑高度
- **网格生成**：从 2D 轮廓拉伸为 3D 模型
- **放置对齐**：将建筑模型精确放置在地形表面

## 当前状态

建筑系统正在开发中，已完成基础的轮廓解析与网格生成。
