# 数据流程

本文档描述克隆地球项目中，从原始地理数据到引擎可用资产的完整数据处理流程。

## 概述

克隆地球的数据流程覆盖了数据获取、预处理、瓦片切割、格式转换等环节。

## 数据流程概览

```
原始数据获取 → 数据清洗 → 坐标投影 → 瓦片切割 → 格式转换 → 引擎加载
```

## 主要数据源

| 数据类型 | 来源 | 格式 |
|----------|------|------|
| 高程数据 | SRTM / ASTER | GeoTIFF |
| 建筑轮廓 | OpenStreetMap | PBF / GeoJSON |
| 道路网络 | OpenStreetMap | PBF / GeoJSON |
| 卫星影像 | 公开卫星数据 | GeoTIFF |

## 当前状态

数据流程的基础框架已搭建，正在针对建筑和道路数据进行优化。
